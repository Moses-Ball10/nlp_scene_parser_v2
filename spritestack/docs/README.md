## What is SpriteStack Studio?

SpriteStack Studio is a desktop pixel-art editor **and** sprite-stacking tool for Windows.  
It combines the drawing/animation workflow of tools like _Aseprite_ with the 3D stacking preview and rotation-sheet export of _SpriteStack Studio_ — all in one application.

### Who is it for?

- **Pixel artists** who want layer-based drawing with animation support
- **Game developers** who use sprite stacking to fake 3D in 2D engines
- **Hobbyists** who want to create pixel art, animate it, and see it in 3D instantly
