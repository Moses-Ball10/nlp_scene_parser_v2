9# Keyboard Shortcuts Reference

Printable quick-reference for all keyboard shortcuts in SpriteStack Studio.

---

## Tools

| Shortcut | Tool                      |
| -------- | ------------------------- |
| `B`      | Pencil (Brush)            |
| `E`      | Eraser                    |
| `G`      | Fill (Bucket)             |
| `I`      | Eyedropper (Color Picker) |
| `L`      | Line                      |
| `R`      | Rectangle                 |
| `C`      | Circle / Ellipse          |
| `S`      | Select                    |

---

## File Operations

| Shortcut       | Action           |
| -------------- | ---------------- |
| `Ctrl+N`       | New Canvas       |
| `Ctrl+O`       | Open Project     |
| `Ctrl+S`       | Save Project     |
| `Ctrl+Shift+S` | Save Project As… |
| `Ctrl+E`       | Export…          |
| `Ctrl+Q`       | Quit             |

---

## Edit

| Shortcut | Action                  |
| -------- | ----------------------- |
| `Ctrl+Z` | Undo                    |
| `Ctrl+Y` | Redo                    |
| `Delete` | Clear Selection / Layer |

---

## View & Navigation

| Shortcut            | Action        |
| ------------------- | ------------- |
| `Ctrl+=`            | Zoom In       |
| `Ctrl+-`            | Zoom Out      |
| `Ctrl+0`            | Fit to Window |
| `Space + Drag`      | Pan Canvas    |
| `Middle-Mouse Drag` | Pan Canvas    |
| `Ctrl+Mouse Wheel`  | Zoom In/Out   |

---

## Animation

| Shortcut | Action                   |
| -------- | ------------------------ |
| `F5`     | Play / Pause             |
| `F6`     | Stop (rewind to frame 1) |
| `F7`     | Next Frame               |
| `F8`     | Previous Frame           |

---

## 3D Preview

| Input                        | Action       |
| ---------------------------- | ------------ |
| `Left-Mouse Drag`            | Rotate model |
| `Left-Mouse Drag (vertical)` | Tilt model   |
| `Mouse Wheel`                | Zoom in/out  |
| `Middle-Mouse Drag`          | Pan view     |

---

## Canvas Drawing

| Input                     | Action                               |
| ------------------------- | ------------------------------------ |
| `Left-Click + Drag`       | Draw with active tool                |
| `Right-Click` (on canvas) | —                                    |
| Brush Size 1–64           | Adjustable via slider in Tools panel |
| Mirror X / Mirror Y       | Toggle via checkboxes in Tools panel |

---

_Print this page and keep it next to your monitor until the shortcuts become second nature!_
